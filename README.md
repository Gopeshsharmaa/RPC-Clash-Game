## 🎮 How to Play  
1. Select **Rock**, **Paper**, or **Scissors**.  
2. The computer will make a random choice.  
3. The winner is determined by the standard rules:  
   - 🪨 Rock beats Scissors  
   - ✂️ Scissors beats Paper  
   - 📜 Paper beats Rock  
4. Play again and try to beat the AI!
